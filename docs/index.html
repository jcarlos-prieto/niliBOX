Discover niliBOX

nilum, nili [n.] - latin: nothing, nothingness, which does not exist.

A software to manage external devices connected locally or remotely.

niliBOX is the evolution of PCR AnyWhere, a well known software to control Icom radio scanners.

niliBOX has been enhanced with new devices, a brand new interface and support for mobiles devices.

niliBOX is at version 1.2.0 (build 2025.07.14). Download it now!

It has been fully refurbished and optimized, specially the graphics, remote access and multi-instance. It is also available for mobile devices (Android and iOS).

For more information, check the release notes and the video tutorials.

And niliBOX is now Open Source!. Check the repository.
